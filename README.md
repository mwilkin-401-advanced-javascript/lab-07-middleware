### Setup
#### `.env` requirements
* `PORT` - if none given the default is 3000

#### Running the app
* `npm start` Starts server
* Endpoint: `/a`
  * Returns current time.
* Endpoint: `/b`
  * Returns a numbered squared.
* Endpoint: `/c`
  * Returns Route C.
* Endpoint: `/d`
  * Raises and error and points to 404.
* Endpoint: `/any`
  * A catch-all routes for unknown routes.
